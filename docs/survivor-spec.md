# 真夏サバイバー 仕様書

## 企画の芯

- ジャンル: `ヴァンパイアサバイバー` 風の見下ろし型オート攻撃サバイバル
- 方向性: ストーリーは薄め、ゲーム性は濃い
- 1プレイ: `20分`
- 勝利条件: `20:00` を生き残る
- 基本操作: `移動のみ`
- 強化: レベルアップ時に `3択`
- メタ進行: コインによる恒久強化、武器・スタイル・ステージ解放

## コンセプト

- 舞台は `終わらない真夏の町`
- 細かい説明はしない
- プレイヤーは、暑さ・噂・町の怪異を相手にとにかく生き残る
- プレイアブルは `先輩` ひとりだけ
- 1キャラで長く遊ばせるため、差別化は `武器構成` `スタイル` `恒久強化` `難易度` `実績` `週替わりルール` で作る
- 物語は最小限:
  - 開始前に1行
  - スタイル解放時に短文
  - 20分クリア時に短い締め

## 500時間向けの柱

- `1キャラでも毎回ビルドが変わる`
- `ステージごとに攻略の正解が変わる`
- `危険度ランク` と `逆境条件` で難度が伸び続ける
- `武器熟練度` と `実績` で中長期目標を切らさない
- `日替わり / 週替わり` の固定条件ランを用意する
- `エンドレス` と `スコアアタック` で理論値詰めができる
- `隠し数字イベント` と `超低確率の裏報酬` で探索欲を維持する

## コアループ

1. ステージを選ぶ
2. 先輩の `開始スタイル` を選ぶ
3. 自動攻撃で敵を倒し、経験値を回収する
4. レベルアップで武器またはパッシブを選ぶ
5. ウェーブに応じて敵の種類と密度が変わる
6. エリートや中ボスから宝箱を獲得する
7. 武器進化や特殊シナジーを完成させる
8. `20:00` 生存または最終ボス撃破を目指す
9. コイン・熟練度・実績進行・危険度解放を持ち帰る

## ラン仕様

### 基本ルール

- 攻撃は自動
- エイムは `最も近い敵` `ランダム` `周囲回転` `一定周期発動` を武器ごとに使い分ける
- 被弾するとHP減少
- HPが0で敗北
- 経験値ジェム、回復アイテム、宝箱、磁石アイテム、コイン袋がドロップ
- 基本ランは `20分`
- 20分クリア後は `エンドレス継続` を選択可能

### 先輩 1キャラ設計

- 名前: `先輩`
- 役割: 近距離寄りの万能型
- 初期武器: `先輩タックル`
- 固有特性: `やりますねえ`
- 基礎ステータス:
  - HP `120`
  - 移動速度 `1.0`
  - 攻撃力倍率 `1.0`
  - クールダウン倍率 `1.0`
  - 回収範囲 `1.0`
  - 防御 `0`

### 先輩の開始スタイル

- プレイアブルは1人だけだが、開始時にスタイルを選べる
- スタイルはキャラ替えではなく `構え` や `育成方針` に近い

#### 1. 通常

- バランス型
- 初心者向け

#### 2. 圧強め

- 近距離火力 `+12%`
- 移動速度 `-5%`
- `やりますねえゲージ` が溜まりやすい

#### 3. 面倒見

- 回復アイテム効率 `+20%`
- ノックバック `+10%`
- 序盤安定型

#### 4. 本気

- ボスダメージ `+18%`
- 雑魚処理補正 `-8%`
- 高難度向け

#### 5. 無口

- クリティカル率 `+8%`
- 取得コイン `+15%`
- 長時間周回向け

### レベルアップ仕様

- 毎回 `3択`
- メタ解放後は `Reroll / Skip / Banish / Lock` を使用可能
- 候補の優先順:
  - 未所持武器
  - 所持武器の強化
  - 未所持パッシブ
  - 所持パッシブの強化
  - 特殊オファー

### レベルアップ候補の詳細

- 武器所持上限: `6枠`
- パッシブ所持上限: `6枠`
- 候補重み:
  - 未所持武器 `35`
  - 所持武器強化 `32`
  - 未所持パッシブ `22`
  - 所持パッシブ強化 `22`
  - 特殊オファー `6`
- 特殊オファー例:
  - 即時回復
  - 宝箱品質上昇
  - 次の3レベルだけ特定カテゴリ出現率上昇
  - `やりますねえゲージ` 即時50%

### XPカーブ

- レベル1→2は軽くする
- 中盤から緩やかに必要量が増える
- 目安式:
  - `必要XP = 5 + floor(level ^ 1.28 * 2.4)`
- 体感:
  - 序盤はテンポよく成長
  - 中盤でビルド完成が見え始める
  - 後半は伸びを厳選する段階

### レベル帯の意味

- `Lv1-10`
  - 初期武器の安定化
  - 範囲処理手段の確保
- `Lv11-25`
  - ビルドの骨格形成
  - パッシブの方向性確定
- `Lv26-40`
  - 進化条件の回収
  - ボス戦向け補強
- `Lv41-60`
  - 完成ビルド化
  - 余剰候補の圧縮
- `Lv61+`
  - エンドレスやスコア向け微調整
  - 余剰成長はコインや小回復へ変換

### レベル上限後の変換

- 全武器・パッシブ最大時:
  - レベルアップ候補は `コイン`, `回復`, `一時バフ` に変換
- これにより後半でも無駄レベル感を減らす

### 宝箱仕様

- エリート敵・中ボス・時間イベント敵からドロップ
- 宝箱等級:
  - `通常箱`
  - `豪華箱`
  - `進化箱`
- 宝箱の役割:
  - レベル上昇
  - 武器進化
  - スタイル専用強化
  - ごく低確率の裏報酬

## 敵ウェーブ仕様

### 設計方針

- `敵の数` だけでなく `速度`, `耐久`, `弾持ち`, `押し込み力`, `特殊能力` で圧を変える
- プレイヤーのビルド進行に合わせて、要求する対策を段階的に変える
- 同じステージでも危険度ランクでウェーブ内容を差し替える

### 基本敵カテゴリ

- `ラッシュ雑魚`
  - 数で迫る
  - 低HP
- `壁役`
  - 高HP
  - 移動遅め
- `追跡型`
  - 移動が速い
  - プレイヤーへ圧をかける
- `遠隔型`
  - 弾または設置物を使う
- `妨害型`
  - 鈍化、視界妨害、回収阻害
- `エリート`
  - 宝箱持ち
- `ボス`
  - 固有行動あり

### 20分ランの標準ウェーブ表

| 時間 | 主な敵 | 意図 |
|---|---|---|
| `00:00-01:30` | セミ群、熱だまり小 | 初期武器の感触確認 |
| `01:30-03:00` | セミ群増量、壁役少量 | 範囲火力の必要性を見せる |
| `03:00` | 初エリート | 宝箱導入 |
| `03:00-05:00` | ラッシュ雑魚 + 追跡型 | 足を止める危険を学習させる |
| `05:00-06:30` | 密集ラッシュ | 雑魚処理力チェック |
| `06:30-08:10` | 遠隔型少量混入 | 立ち位置の工夫が必要になる |
| `08:10` | 数字イベント湧き | 810イベント、特別報酬候補 |
| `08:10-10:00` | 妨害型 + 壁役 | 中盤ビルドの穴を突く |
| `10:00` | 中ボス1 | 進化準備の確認 |
| `10:00-12:30` | 追跡型増量、遠隔型増量 | 単一火力と機動力の両立を要求 |
| `12:30-15:44` | 複合編成 | ビルド完成度チェック |
| `15:44` | 危険フェーズ | 画面圧最大、時計針系追加 |
| `15:44-18:00` | 高密度ラッシュ + エリート多発 | 終盤用の完成ビルド前提 |
| `18:00-20:00` | ボス護衛群 + 妨害型 | 最終準備 |
| `20:00` | 最終ボス | クリア判定 |

### 危険度ランク

- ステージごとに `危険度0-15`
- 各ランクで以下が強化:
  - 敵HP
  - 敵移動速度
  - 遠隔弾密度
  - エリート数
  - ボスの追加行動
- `危険度8` 以降で新敵カテゴリ解放
- `危険度12` 以降でウェーブの順序自体が一部変化

### ウェーブの変調ルール

- 毎ラン完全固定ではなく、各時間帯に `候補群` を持つ
- 例:
  - `05:00-06:30` は `セミラッシュ型`, `熱だまり型`, `混成型` の3候補
- これで暗記を防ぎつつ、理不尽なランダムにはしない

### 8:10 イベント

- `08:10` で発生する特別イベント
- 候補:
  - ボーナス経験値群
  - 特殊宝箱持ちエリート
  - コイン大量湧き
  - 隠し実績条件敵
- 低確率で `810専用の裏敵` が出現

### 15:44 危険フェーズ

- BGM、敵色、背景演出を変える
- 敵の押し込み力を大きく上げる
- 新ギミック:
  - 時計針弾
  - 一時停止フィールド
  - 画面端からの圧縮湧き
- プレイヤーが完成ビルドでないと一気に崩れる山場

### エンドレス

- 20分クリア後に選択制
- 以降は2分ごとに危険度が自動上昇
- 伸び続ける指標:
  - スコア
  - 撃破数
  - 最大連続回避時間
  - `やりますねえ` 発動回数

## 長期やり込み仕様

### 1. 武器熟練度

- 12武器それぞれに熟練度
- 熟練度上限は `50`
- 報酬:
  - 微強化
  - 武器スキン
  - 進化演出
  - 専用実績

### 2. スタイル熟練度

- 先輩の開始スタイルごとに熟練度
- 各スタイルに独自ミッション

### 3. 実績ボード

- 通常実績
- 変態実績:
  - 被弾0で20分
  - `8:10` イベントを10回成功
  - `15:44` を低レベルで突破
  - `やりますねえ` を1ランで20回発動

### 4. 危険度制覇

- 各ステージで危険度15まで突破
- さらに `逆位相モード` 解放

### 5. 週替わりルール

- 武器候補制限
- パッシブ固定
- 敵強化
- スコアランキング

### 6. 日替わりシード

- 同条件で全プレイヤーが遊べる
- スコア競争向け

### 7. 裏報酬

- `810` と `114514` に絡む極低確率イベント
- 実績名、演出、スキン、称号に使う
- 直接の強課金要素や勝率破壊要素にはしない
