## 第1章 真夏町にて

- 主人公が終わらない午後の町へ迷い込む
- 駅前で住民から「怖いのに優しい先輩」の噂を聞く
- `810` の数字が駅前から学校まで不自然に残っている
- 学校のプールで `熱だまり` を鎮め、先輩と初対面

主なセリフ:

```txt
主人公「暑い……というか、夏が終わる気配がない」
アイス屋の子「先輩、こわい顔してるのに当たり棒くれたよ」
サラリーマン「あの人に見られると体感温度が3度上がる」
おばちゃん「でも犬には好かれてるのよ」
監視員「壊さずに済むなら、そのほうがいい」
先輩「お前、熱中症なりかけてるだろ」
主人公「噂と違って、最初の言葉が優しい」
先輩「噂なんてだいたい雑だ」
```

## 第2章 商店街の影

- 先輩に頼まれて、商店街で広がる妙な噂を回収する
- 噂は熱に触れて形を持ち始めている
- `うわさかぶり` と戦い、言葉が敵になる感覚を学ぶ

主なセリフ:

```txt
先輩「聞きっぱなしにすると、噂は勝手に育つ」
駄菓子屋「この町じゃ、言葉も汗をかくのよ」
セミ博士「鳴き声と噂話はよく似ている。止まらないとこがね」
主人公「似てるけど、今それどころじゃない」
うわさかぶり「コワイ　コワイ　コワイ」
主人公「それ、お前の声か。みんなの声か」
先輩「見た目より中身を見ろ。難しいけどな」
```

## 第3章 プールサイドの秘密

- 学校のプール脇に隠された保健記録を調べる
- 町の暑さは自然現象ではなく、誰かの未練と結びついていると知る
- `塩素まぼろし` を鎮める

主なセリフ:

```txt
保健委員「倒れた人の記録、毎年ここで終わってるんです」
主人公「毎年？」
監視員「この町では、“毎年”が同じ日に重なる」
先輩「終われないものが、熱になる」
主人公「じゃあ、先輩は何を抱えてるんです」
先輩「抱えてるから、まだここにいる」
```

## 第4章 15:44の駅

- 止まった時計だらけの駅へ向かう
- 電車が来ない理由と、時間が 15:44 に止まった理由を探る
- `止まった改札` との戦闘で「待つこと」がテーマになる

主なセリフ:

```txt
駅のアナウンス「まもなく列車が……まいりません」
主人公「言い切った」
サラリーマン「待ち続けると、人は景色になる」
先輩「昔、この駅で帰れなかったやつがいた」
主人公「……先輩のことですか」
先輩「半分はな」
改札の残響「マダ　マダ　マダ」
主人公「待つだけじゃ進まない。でも、見捨てるのも違う」
```

## 第5章 夏祭りの残響

- 神社と夜店通りに残る、終わらなかった祭りの音を追う
- `810` の拍と祭りばやしが章ギミックになる
- 町の噂が楽しさと寂しさの両方を抱えていたことが見える
- `祭囃子のぬけがら` と戦う

主なセリフ:

```txt
太鼓娘「鳴らない祭りばやしほど、耳に残るんだよ」
迷子の子「帰るって、どこに？」
主人公「その質問、重いな……」
先輩「楽しかった場所ほど、終わりを認めにくい」
主人公「先輩も？」
先輩「……そうだ」
祭囃子のぬけがら「マダ　オワリタクナイ」
主人公「終わることと、消えることは違うだろ」
```

## 第6章 真昼の獣

- 先輩への恐怖や誤解が町全体で膨らみ、ついに怪物の姿になる
- ここでプレイヤーの `kindness / fear` が強く反映される
- `真昼の獣` は先輩本人ではなく、町が作った“怖さの像”

主なセリフ:

```txt
町の人「見たか、あれが本当の先輩だ」
主人公「違う。これは噂が勝手に膨らんだだけだ」
監視員「ようやく見えたか」
先輩「俺はあんなに器用じゃない」
主人公「自分で言います？」
真昼の獣「コワイ　ホラ　ニゲロ」
主人公「逃げない。ちゃんと見る」
先輩「……助かる」
```

## 第7章 夕立の前で

- プールの底と駅の時間がつながる最深部へ
- `114514` の最終認証を越え、先輩の正体が明らかになる
- 先輩は「最初にこの町へ取り残された水泳部の先輩」だと分かる
- `終わらない午後` と向き合い、夏を終わらせるか決める
- クリア

主なセリフ:

```txt
先輩「ここまで来たら、もう誤魔化せない」
主人公「終わらない夏の正体」
先輩「俺は、この町で最初に帰れなかった“先輩”だった」
先輩「帰れなかった気持ちの寄せ集めだ」
主人公「じゃあ、終わらせよう」
終わらない午後「オワレナイ　オワラセナイ」
主人公「終わっても、なくならないものはある」
先輩「……それを言えるなら、大丈夫だ」
主人公「先輩も、一緒に帰りましょう」
```

## エンディングの方向

- `kindness` が高い
  - 夏が夕立でほどけ、先輩も町も前へ進む
- `fear` が高い
  - 夏は終わるが、少し苦い別れになる
- どちらでも第7章でゲームはクリア
